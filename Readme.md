# Task Manager Full Project

This archive contains a complete task manager project:

- backend/  -> Node + Express + MongoDB (JWT auth)
- frontend/ -> React + Material UI frontend (Create React App)

Quick start (backend):
1. cd backend
2. cp .env.example .env (edit if needed)
3. npm install
4. npm run dev

Quick start (frontend):
1. cd frontend
2. npm install
3. npm starts

Make sure MongoDB is running locally or change MONGO_URI in backend/.env.
